# ⚙️ CONCEPT_LOGIC (개념 및 시스템 로직)

## 1. AI 펫 튜터: 소크라테스식 힌팅 (Socratic Hinting)
단순한 정답 제시가 아닌, 학습자가 스스로 생각하게 유도하는 AI 에이전트 로직입니다.

### 🧠 로직 순서
1. **트리거**: 사용자가 오답을 입력하거나 30초 이상 무반응일 때.
2. **맥락 파악**: 현재 문제 유형(예: 분수의 덧셈)과 오답 패턴 분석.
3. **힌트 생성 (LLM)**:
    - **Step 1 (연상)**: "피자 조각이 총 몇 개인지 볼까?" (시각적 비유)
    - **Step 2 (원리)**: "분모가 다르면 바로 더할 수 없지. 친구들 이름을 맞춰줄까?" (통분 유도)
    - **Step 3 (응원)**: "거의 다 왔어! 조금만 더 생각해보자!"
4. **출력**: 펫의 말풍선 UI로 텍스트 렌더링.

## 2. URL 기반 무결성 동기화 (Stateless Sync)
별도의 데이터베이스 없이 사용자의 모든 상태를 URL에 담아 유지하는 혁신적인 로직입니다.

### 🛰️ 데이터 구조 (State JSON)
```json
{
  "u": "UserNickname",
  "c": 1250, // Coins
  "p": { "id": "cat_01", "lv": 3, "f": 80 }, // Pet state
  "h": "g2_a1,g2_a2", // Learning history (grade2_arithmetic1, etc.)
  "r": "f_01,f_05" // Room items (furniture ids)
}
```
### 🔄 변환 프로세스
1. **직렬화**: State 객체를 JSON 문자열로 변환.
2. **압축**: `LZ-String` 알고리즘을 사용하여 문자열 길이를 70% 이상 단축.
3. **URL 주입**: `window.history.replaceState`를 통해 `?s=BASE64_STRING` 형태로 URL 갱신.
4. **공유**: 사용자는 이 URL을 북마크하거나 나에게 보내기 등으로 상태를 영구 보존.

## 3. 3D 인터랙션 엔진 (React Three Fiber)
고학년 기하학 학습을 위한 WebGPU/WebGL 렌더링 로직입니다.

### 📐 주요 구현체
- **CylinderSlicer**: 원기둥을 가로/세로로 잘랐을 때의 단면 모양을 실시간으로 계산하여 시각화.
- **PolyhedronFolder**: 전개도를 접어서 입체도형이 되는 과정을 슬라이더로 조절하여 관찰.
- **RotationalSolidGen**: 평면도형을 축을 중심으로 회전시켜 회전체가 생성되는 궤적 렌더링.

## 4. 리텐션 및 보상 경제 (Economy Loop)
- **데일리 스탬프**: 7일 연속 출석 시 한정판 펫 스킨 지급.
- **퀘스트 보드**: "오늘의 연산 10문제 풀기" 등 단기 목표 제시로 도파민 보상 관리.
- **학습 인증 카드**: HTML-to-Canvas 기술을 활용하여 공유용 이미지 자동 생성.
