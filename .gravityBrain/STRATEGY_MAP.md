# 🗺️ STRATEGY_MAP (중장기 전략 지도)

## 🎯 비전 (Vision)
"공식이 아닌 감각으로, 아이들의 첫 수학을 가장 아름답게 만드는 플랫폼"

## 🚀 전략 방향 (Strategic Pillars)

### 1. 시각적 압도감 (Visual Mastery)
- **3D 원리 도구**: Three.js를 활용한 고학년 기하학 학습 도구 고도화.
- **마이크로 애니메이션**: 모든 상호작용에 학습 동기를 부여하는 부드러운 피드백 추가.

### 2. 지능형 동반자 (Intelligent Companion)
- **AI 펫 튜터**: 소크라테스식 문답법으로 정답이 아닌 '생각의 길'을 제시하는 LLM 기반 힌트 시스템.
- **적응형 난이도**: 학습 성취도에 따라 실시간으로 변화하는 맞춤형 문제 흐름.

### 3. 무경계 동기화 (Seamless Sync)
- **URL 기반 상태 공유**: 로그인 없이도 자신의 학습 데이터(펫, 코인, 진도)를 URL 하나로 영구 보존 및 기기 간 이동.
- **비주얼 인증 카드**: 인스타그램/카카오톡 등 SNS 공유에 최적화된 학습 결과 자동 렌더링.
