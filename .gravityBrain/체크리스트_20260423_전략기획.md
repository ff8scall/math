# 체크리스트: 통합 전략 기획 및 서비스 전면 점검 (2026-04-23)

## 1. 단계: 탐색 (Research)
- [ ] 현재 시장 내 초등 수학 에듀테인먼트 경쟁 서비스 분석
- [ ] 매쓰 펫토리만의 독보적인 가치 제안(UVP) 도출
- [ ] 타겟 사용자(초등 저학년~고학년)의 학습 패턴 및 니즈 분석

## 2. 단계: 구체화 (Brainstorm)
- [ ] '수학 원리' 시각화 도구의 고도화 아이디어 (현재 2D -> 3D 확장 등)
- [ ] 펫 키우기 시스템의 게임화 요소 강화 (퀘스트, 보상, 커스터마이징)
- [ ] 기술적 한계 및 기술 스택 최적화 방안 도출

## 3. 단계: 성장 (Monetize/Growth) - *수익화 대신 지속 가능성에 집중*
- [ ] 리텐션 확보를 위한 데일리 루프 설계
- [ ] 바이럴 및 공유 기능 고도화 (학습 결과표 공유 등)
- [ ] 커뮤니티 요소 도입 검토 (친구 펫 방문 등)

## 4. 단계: 여정 (User Journey)
- [ ] 신규 사용자의 온보딩(Initial Experience) 최적화
- [ ] 학습 -> 보상 -> 펫 성장으로 이어지는 사용자 흐름(Flow) 정교화
- [ ] 이탈 포인트(Pain Points) 예측 및 해결책 마련

## 5. 단계: 검증 (Pre-verify)
- [ ] 브라우저 성능 리스크 (Framer Motion 다량 사용 시 부하 등) 점검
- [ ] 데이터 저장 안정성 (localStorage 한계) 및 동기화 리스크 분석
- [ ] 콘텐츠 정합성 및 교육적 효과 검증 설계

## 6. 단계: 확정 (MVP Scope)
- [ ] Phase 2: 핵심 보완 기능 우선순위 선정
- [ ] 개발 로드맵 수립 및 마일스톤 설정
- [ ] 최종 기획 문서(`STRATEGY_MAP`, `CONCEPT_LOGIC` 등) 일괄 갱신

---
**진행 가이드**: 각 단계 완료 시마다 사용자에게 진행 상황을 보고하고, 다음 단계 진입 전 승인을 득함.
