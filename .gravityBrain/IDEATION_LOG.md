# 📔 IDEATION_LOG (기획 로그)

## 📅 2026-04-23: Phase 2 통합 전략 기획 회의

### 📝 결정 사항 1: 데이터 저장 방식의 전환
- **이슈**: LocalStorage만 사용 시 기기 이동이나 브라우저 초기화 시 데이터 유실 리스크가 큼.
- **결정**: 서버리스(Serverless) 지향을 유지하기 위해 별도 DB 대신 **URL State Sync (LZ-String)** 방식을 채택함.
- **이유**: 사용자에게 가입의 피로감을 주지 않으면서도 자신의 상태를 북마크나 링크 공유로 영구 보존할 수 있는 가장 가볍고 효율적인 방식임.

### 📝 결정 사항 2: 고학년 타겟팅 확장
- **이슈**: 현재 연산 위주의 콘텐츠는 고학년 사용자에게 매력이 떨어짐.
- **결정**: Three.js를 활용한 **3D 기하학 실험실**을 핵심 콘텐츠로 추가함.
- **이유**: 6학년 수학의 복잡한 입체도형 원리를 2D 평면이 아닌 3D로 직접 만져보게 함으로써 타 학습 서비스와 차별화된 '압도적 도구' 이미지를 구축함.

### 📝 결정 사항 3: AI의 역할 정의
- **이슈**: 정답만 알려주는 AI는 교육적으로 유해할 수 있음.
- **결정**: 펫을 페르소나로 한 **'소크라테스식 힌트 봇'**으로 정의함.
- **이유**: 아이가 펫과 대화하며 스스로 답을 찾아가게 함으로써, 학습 효과와 정서적 유대감을 동시에 잡음.

### 📝 결정 사항 4: 성장을 위한 공유 루프
- **이슈**: 검색 유입(SEO) 외에 사용자 간 전파가 부족함.
- **결정**: 학습 완료 시 **'비주얼 인증 카드'** 자동 생성 기능 도입.
- **이유**: 아이들이 자신의 성취(펫 성장, 뱃지 등)를 자랑하고 싶어 하는 심리를 이용하여, 부모님들의 SNS를 통한 자연스러운 서비스 노출 유도.
